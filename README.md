# Python Project - DecodeLabs Industrial Training Kit

A collection of three beginner-friendly Python projects designed to teach core programming concepts and real-world application development.

## 📚 Projects

### Project 1: To-Do List
**Goal:** Build a program where users can add tasks to a list and view them.

**Key Skills:**
- Python lists (`append()`)
- Loops (`for` loops)
- String manipulation
- User input handling

**Features:**
- Add new tasks
- View all tasks with numbering
- Remove specific tasks
- Clear all tasks
- Menu-driven interface

**Run:**
```bash
python project1.py
```

---

### Project 2: Expense Tracker
**Goal:** Create a script where users enter expense amounts and the program displays the total spent.

**Key Skills:**
- Math operations
- Accumulators (`total = total + new_expense`)
- Data structure management
- Summary statistics

**Features:**
- Add expenses with optional notes
- View detailed expense list
- View running total
- View summary (count, average, highest, lowest)
- Remove individual expenses
- Clear all expenses

**Run:**
```bash
python project2.py
```

---

### Project 3: Random Password Generator
**Goal:** Write a tool that asks for a password length and generates a random, complex password using letters and numbers.

**Key Skills:**
- Importing modules (`random`, `string`)
- String manipulation
- List comprehension
- Input validation

**Features:**
- User-defined password length (4-128 characters)
- Optional special characters
- Random generation using Python's `random` module
- Input validation and error handling

**Run:**
```bash
python project3.py
```
